Ampify
======

.. class:: big

The open source Ampify Platform provides you with a web application framework
that enables you to create social apps on top of a secure, decentralised core.

We are working towards a first working release for the end of 2010. If you've
already read the `building Ampify <building-ampify.html>`_ guide, run the
following to build the current code:

.. syntax:: bash

  $ git clone https://github.com/tav/ampify.git
  $ cd ampify
  $ make
  $ . environ/ampenv.sh

Contribute
----------

To contribute any patches, simply fork `the repository
<http://github.com/tav/ampify>`_ using GitHub, create a new branch with your
patch, add yourself to the ``AUTHORS`` file and then run ``git review submit``,
e.g.

.. syntax:: bash

  # Create a new branch for your patch:
  $ git checkout -b name-for-your-patch

  # Make the various changes you want to make and
  # submit it to be to be reviewed by running:
  $ git review submit

  # Make any changes and submit updates until the
  # reviewers give a LGTM (Looks Good To Me) reply
  # on http://gitreview.com -- that's all!
